# Magway PCB Package

This folder contains the current Î¼Touch/Magway board design exported from Altium. PCB design by [Xiaomeng Chen](https://github.com/blankchenxm).

## Contents
- `Magway.PrjPcb` / `Magway.PrjPcbStructure`: project files.
- `Magway.SchDoc`, `Magway.SchLib`: schematic and symbol library.
- `Magway.PcbDoc`, `Magway.PcbLib`: layout and footprint library.
- `Magway.BomDoc`: bill of materials.
- `PnPout/`: pick-and-place HTML output.
- `Project Outputs for Magway/`: DRC reports.
- `Project Logs for Magway/`: ECO log.
- `__Previews/`: Altium preview assets.
- `History/`: project history metadata.

## Tooling
Designed in Altium Designer (original version 20.x). Use the `.PrjPcb` entry point to open the project; outputs and previews are included for reference.
